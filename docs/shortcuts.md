# Keyboard Shortcuts

| Shortcut | Context | Action |
|----------|---------|--------|
| `Ctrl/Cmd + K` | Global | Open command palette |
| `Ctrl/Cmd + S` | Document builder | Save |
| `Ctrl/Cmd + P` | Document builder | Preview (after save) |
| `Ctrl/Cmd + D` | Document builder | Duplicate |
| `Delete` | Document builder (empty field) | Remove item row |

## Command Palette

Press `Ctrl/Cmd + K` from anywhere to open the command palette, which provides quick access to:

- All pages (Dashboard, Documents, Customers, Products, Analytics, Settings)
- Create new documents of any type
